## 10. Animation Timeline

1. **Page Load** (0-1s):
   - Header slides down
   - Search bar fades in
   - Cards appear one by one (staggered)

2. **Card Click** (0.3s):
   - Expand icon rotates 180Â°
   - Card expands smoothly
   - Content fades in

3. **Theme Toggle** (0.3s):
   - All colors transition
   - No jarring changes
   - Smooth fade

4. **Hover Effects** (0.15s):
   - Cards lift up 5px
   - Shadow increases
   - Quick, snappy

## Key Design Principles

1. **Neumorphism**: Soft, 3D-like shadows
2. **Material Design**: Elevation and depth
3. **Smooth Transitions**: Everything animated
4. **Color Consistency**: Blue-gold theme
5. **Icon Usage**: Emoji for visual clarity
6. **Whitespace**: Generous padding
7. **Typography**: Clear hierarchy
8. **Accessibility**: High contrast, focus states

## Mobile Optimizations

- Touch-friendly buttons (min 44px)
- Simplified header on mobile
- Single column layout
- Larger font sizes
- Easy-to-tap cards
- No hover effects (tap instead)
- Swipe-friendly
